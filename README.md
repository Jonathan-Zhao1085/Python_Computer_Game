Setup Instructions:
  1. download all files to "Windows (C:)"
  2. run "main.py"

Description:
  - A two player game of tag, whoever reaches a score of 5 first wins
  - Controls can be found in the "players" menu
  - Player data is saved through reading/writing to a text file
